# Car Database Analysis

This project demonstrates structured data analysis using SQL and MySQL.
It includes database exploration, query design, joins, filtering, and aggregations to extract meaningful insights from a car dataset.

## Tools Used
- MySQL
- SQL

## Key Features
- Created and managed relational database tables
- Designed and executed SQL queries with JOINs
- Applied filtering and aggregation techniques
- Interpreted results for actionable insights

## Objective
To analyze a car dataset with SQL and demonstrate practical data querying skills.
